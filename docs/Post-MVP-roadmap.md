
### **Post-MVP Roadmap**

**1. Full Account Integration & Synchronization**
* **Google Account Login:** Implement full Google OAuth2 authentication to allow users to connect their YouTube accounts.
* **Multi-Account Support:** Allow users to be logged into and switch between multiple YouTube accounts.
* **Local User Profiles:** Add a system for multiple local profiles, allowing different users on the same computer to have separate libraries.
* **Advanced Data Synchronization:** Build the detailed sync engine from the Settings page to sync and manage:
    * Private & Unlisted Playlists
    * Liked Videos
    * Watch History
    * Comments
* **Sidebar Integration:** Add "Liked Videos" and "Watch Later" to the main sidebar navigation, which will become functional upon login.

**2. Playlist & Organization Enhancements**
* **Playlist Folders:** Allow users to group their playlists into custom folders for better organization.
* **Custom Tagging System:** Implement a feature for users to create and assign their own custom tags to playlists and videos for advanced filtering.
* **Playlist Duplication:** Add the ability to duplicate an entire playlist.
* **Sharing:** Implement a "Share Playlist" feature.

**3. Downloading & Player Upgrades**
* **Advanced Download Resumption:** Upgrade the download manager to support true resuming of interrupted downloads without starting from scratch.
* **Online Streaming Player:** Build a feature to stream videos directly from YouTube without downloading them first, including a quality selector in the player for streams.
* **Advanced Playback Controls:** Add features like playback speed controls to the video player.

**4. UI & UX Polish**
* **Enhanced Dashboard:** Add the real-time "Active Downloads" widget to the main dashboard.
* **Enhanced UI Previews:** Implement the more detailed preview pop-ups when adding new content.
* **User-Selectable Theme:** Allow users to choose their own primary accent color from a list in Settings.
* **Trash Can & Restoration:** Implement the full "Trash" system where deleted items can be viewed and restored before being permanently deleted.
* **Multi-Select & Batch Actions:** Add the ability to select multiple videos or playlists at once to perform actions like delete, move, or duplicate.

**5. Import & Export**
* **Full Backup & Restore:** Implement the feature to export a user's entire library (playlists, settings, etc.) to a single file, and to import that file to fully restore their setup.
